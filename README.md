 Health Tracker App

> A modern Flutter app for tracking weight, BMI, and personal health progress.
> Ứng dụng Flutter giúp theo dõi cân nặng, chỉ số BMI và đặt mục tiêu sức khỏe cá nhân.  
> Designed and developed by Lăng Nhật Minh.

---

Features

- Login / Register / Forgot Password
- Weight tracking with date, notes, and change history
- BMI calculation and real-time classification
- Goal setting (target weight, time estimation, weekly progress)
- Charts showing trends over time
- Smart search bar with debounce
- Modern UI / Dark–Light theme ready
- Firebase Auth & Firestore Integration (optional)
